    CSE Training Workshops in Data Analytics, Fall 2014 • DCL L440, 1–3 pm

    Project maintained by uiuc-cse Hosted on GitHub Pages — Theme by mattgraham

    All workshops will be held in L440 Digital Computer Laboratory, an EWS computer laboratory in the basement. If you are not an engineering student, you will need an EWS login to use the computer lab, so please email Neal Davis (contact information at bottom) at least 24 hours in advance.

    There is no sign-up for this series—walk-ins are welcome and encouraged!

    Introduction to R, Part 1

    Sep. 8, 1:00–3:00 pm • DCL L440

    This workshop targets students with some programming experience and little to no prior exposure to the statistical and data analysis language R. We will conduct a hands-on walkthrough of basic R features and packages.

    We will cover the following topics:

    • Reading and cleaning data
    • Major data types and their uses
    • Useful functions
    • Data display
